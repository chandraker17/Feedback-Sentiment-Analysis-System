- Overview
A Python application for collecting customer feedback, doing sentiment analysis with NLP and showing insights as an interactive dashboard.

- Tech Stack
Python
MySQL
Streamlit
TextBlob
Pandas

- Features
Gather your customer’s feedback in the rating form
Sentiment (Positive/Negative/Neutral) classification
Interactive Streamlit dashboard
Product-wise rating insights

- How to Run
1) git clone https://github.com/your-username/Customer-Feedback-Sentiment-System.git
2) cd Customer-Feedback-Sentiment-System
3) pip install -r requirements. txt
4) streamlit run app.py
